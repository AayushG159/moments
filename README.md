## Installation

Clone the repo:

```
$ git clone https://github.com/greyli/moments.git
$ cd moments
```

Create & activate virtual env then install dependency with PDM:

```
$ pdm install
```

Generate fake data then run:

```
$ pdm run flask forge
$ pdm run flask run
* Running on http://127.0.0.1:5000/
```
